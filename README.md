# vAlerts Bot

Discord bot to serve VATSIM alerts to users

## Supported alerts

- [ ] Traffic around airport
- [ ] Traffic overflying boundary
- [ ] Weather
- [ ] Steamer flights starting or ending at an airport
- [ ] Position gets staffed
- [ ] Position becomes unstaffed
- [ ] It's a specific time range

### Combinations

Alert conditions can be chained together to create a more complicated condition.

**Example**: there's traffic at an airport, the wind is from the east, and it's between 1000 Z and 1200 Z.
